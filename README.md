# Healthcare-Revenue-Patient-Analytics-Dashboard
This project analyzes hospital patient, billing, and department data to identify revenue drivers, insurance contribution patterns, and financial performance trends.
The goal was to simulate a real-world healthcare analytics scenario using Excel for data cleaning and Power BI for modeling and visualization.
